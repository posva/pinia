# SFC Playground

This is continuously deployed at [https://play.pinia.vuejs.org/](https://play.pinia.vuejs.org).

## Run Locally in Dev

In repo root:

```sh
pnpm -F @pinia/online-playground dev
```

## Build for Prod

In repo root

```sh
pnpm -F @pinia/online-playground build
```

Preview the build

```sh
pnpm -F @pinia/online-playground serve
```
